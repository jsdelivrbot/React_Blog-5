# Project Description:
My Blog is a simple blog which you can add a new post or delete an existing post.
# Technical Specifications:
## Tech-Stack:
* React
* Redux
* React-Redux
* axios
* lodash
* React-Router-Dom
* React-Form
* redux-promise
